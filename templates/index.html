Data Configuration Required

MovieLens dataset files not found in system directory

To complete your installation, please download and configure the required dataset files.

Setup Instructions:

  1. Download the MovieLens dataset from GroupLens Research.
    The "small" or "latest-small" dataset version (100K ratings) is recommended for faster processing and demonstration purposes.
  2. Extract the downloaded ZIP file to access the dataset files.
  3. Create a static/data directory inside your RecommendPro application folder.
  4. Copy the ratings.csv and movies.csv files from the extracted folder to the static/data directory.
  5. Restart the application to load the dataset.
Expected Directory Structure:
