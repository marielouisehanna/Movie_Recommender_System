MovieLens dataset files not found!

To use this application, you need to download the MovieLens dataset and place the files in the correct directory.

Instructions:

  1. Download the MovieLens dataset from GroupLens Research. The "small" or "latest-small" version (100K ratings) is recommended for faster processing.
  2. Extract the downloaded ZIP file.
  3. Locate the ratings.csv and movies.csv files in the extracted folder.
  4. Create a static/data directory inside the application folder.
  5. Copy both files into the static/data directory.
  6. Restart the application.
Expected Directory Structure:
your_app_folder/
├── static/
│   ├── data/
│   │   ├── ratings.csv
│   │   └── movies.csv
│   ├── css/
│   └── img/
└── templates/
